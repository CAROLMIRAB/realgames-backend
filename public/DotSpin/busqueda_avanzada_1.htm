1.2.1. Búsqueda Avanzada

 
La sección de Búsquedas avanzadas o modo experto permite realizar la consulta de un usuario mediante la introducción de términos contenidos en el perfil del usuario.
 
 
 

Buscar Usuario

 
Formulario para realizar búsqueda de usuario según el criterio seleccionado.
 
Arriba

Id de usuario

 
Permite realizar búsquedas mediante el id del usuario.
Arriba

DNI

 
Permite realizar búsquedas mediante el documento de Identidad registrado en el sistema.
Arriba

Nombre

 
Permite realizar búsquedas mediante el nombre registrados en el sistema.
Arriba

Apellido

 
Permite realizar búsquedas mediante el apellido registrado en el sistema.
Arriba

Usuario

 
Permite realizar búsquedas mediante el nombre de usuario.
Arriba

Correo

 
Permite realizar búsquedas mediante el correo que posee el usuario registrado en el sistema.
Arriba

Buscar

 
Al presionar el botón buscar será realiza el proceso de búsqueda del usuario según el criterio seleccionado.
 
Arriba

Limpiar

Al presionar en el botón limpiar búsqueda los campos del formulario serán borrados y será eliminado el resultado de la búsqueda anterior.
Arriba

Resultados de la búsqueda

 
Como resultado de la ejecución de una búsqueda, el sistema presenta un listado de los usuarios que cumplen con las condiciones indicadas. El sistema presenta un listado de los usuarios que cumplen con las condiciones indicadas en el formulario. Asimismo se presenta el número de usuarios encontrados que cumplen con el criterio de búsqueda.
 
 
Arriba

Detalles

 
A partir de esta presentación inicial se puede seleccionar el usuario de su interés pinchando en “Detalles”, desde donde podrá consultar la ficha descriptiva del usuario.
