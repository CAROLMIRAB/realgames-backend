1.3.1. Búsqueda Avanzada

 
La sección de Búsquedas avanzadas o modo experto permite realizar la consulta de una transacción mediante la información de la operación.
 
 
 

Buscar Usuario

 
Formulario para realizar búsqueda de transacción según el criterio seleccionado.
 
Arriba

Rango de fecha

 
Permite realizar búsquedas mediante un rango de fecha seleccionado.
Arriba

ID

 
Permite realizar búsquedas mediante el Id de la transacción.
Arriba

Usuario

 
Permite realizar búsquedas mediante el Usuario registrado en el sistema.
Arriba

Id Juego

 
Permite realizar búsquedas mediante el Id del Juego.
Arriba

Juego

 
Permite realizar búsquedas mediante el nombre del Juego.
Arriba

Limpiar

 
Al presionar en el botón limpiar búsqueda los campos del formulario serán borrados y será eliminado el resultado de la búsqueda anterior.
 
Arriba

Buscar

Al presionar el botón buscar será realiza el proceso de búsqueda del usuario según el criterio seleccionado.
Arriba

Resultados de la búsqueda

 
Como resultado de la ejecución de una búsqueda, el sistema presenta un listado de las transacciones que cumplen con las condiciones indicadas. Asimismo se presenta el número de transacciones encontradas que cumplen con el criterio de búsqueda.
